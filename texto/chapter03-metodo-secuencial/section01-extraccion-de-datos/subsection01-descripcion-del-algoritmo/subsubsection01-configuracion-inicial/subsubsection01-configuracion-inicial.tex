\subsubsection{Configuración Inicial}


En esta sección, se establecen los parámetros iniciales y se preparan las estructuras de datos necesarias para el análisis.

\begin{verbatim}
Establecer el umbral de distancia para considerar una interacción.
Leer el archivo PDB para obtener la estructura de la proteína
Inicializar listas para:
    - Residuos de ligando (ligand_residues)
    - Átomos de proteína (protein_atoms)
    - Lista de residuos (residues_list)
\end{verbatim}


\paragraph{Ejemplo:}

Para la proteína \texttt{1znf}, se establece un umbral de distancia de 4.0 \AA \newline (Angstroms \cite{angstrom}). Se lee el archivo \texttt{1znf.pdb} y se inicializan las listas vacías para almacenar los residuos de ligando, átomos de proteína y lista de residuos.
